Web Application Security

SQL Injection

Occurs when malicious SQL statements are inserted into entry fields for execution. Prevent it by using prepared statements (PDO) in PHP.

XSS (Cross-Site Scripting)

Attackers inject client-side scripts into web pages viewed by other users. Prevent it by sanitizing all user inputs and escaping outputs.
