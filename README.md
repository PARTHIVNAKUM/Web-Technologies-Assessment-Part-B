# Student IT Skills Application

A frontend-only web application developed for the MSc Computer Science - Web Technologies module. This project demonstrates core web development skills, including HTML5 structure, CSS3 styling, and JavaScript interactivity.

## Features
*   **Dashboard:** Central hub for navigation.
*   **Skills Matrix:** Dynamic display of technical and soft skills.
*   **Learning Resources:** Consolidated curriculum tracker for Weeks 1-10.
*   **Multimedia:** Integration of video previews and image galleries.
*   **Informational Pages:** Guides on Server Config, Security, and Advanced Web Features.
*   **Contact Form:** JavaScript-validated client-side form.

## Tech Stack
*   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
*   **Data:** JSON (with automatic fallback for local file system access)

## Setup
1.  **Clone/Download** the repository.
2.  **Open** `index.html` in any modern web browser.

## Note
This application uses a robust fallback mechanism to ensure all content loads correctly even without a web server, avoiding CORS issues when opening files directly.
