Server Configuration & Deployment

Overview of the concepts covered in Unit 4 & 5 regarding local server environments.

The WAMP Environment

WAMP stands for Windows, Apache, MySQL, and PHP. It is a software stack that enables local web development.

  • Apache: The web server that processes HTTP requests.
  • MySQL: The database management system.
  • PHP: The server-side scripting language.

Localhost & Port 80

C:\wamp64\www\

This is the root directory where project files must be placed to be served by Apache.

If Port 80 is occupied, Apache config (httpd.conf) must be updated to listen on Port 8080.

Deployment Process

Standard deployment flow used in the module:

  1. Develop HTML/CSS locally.
  2. Start WAMP Server (Ensure icon is Green).
  3. Move project folder to www directory.
  4. Access via browser at http://localhost/ProjectName.
